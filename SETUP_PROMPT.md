# 🚀 נוסח להדבקה ב-Claude Code

מעתיקים את כל הבלוק שבין הקווים ומדביקים ב-Claude Code (בתיקייה ריקה חדשה).
Claude יעשה את **כל** העבודה הטכנית. יישארו לך רק 3 דברים שרק את יכולה לעשות:
התחברות לחשבונות, הזנת אמצעי תשלום ב-Railway (~5$ לחודש), וסריקת QR מהטלפון.

> דרישות מוקדמות (חינם): חשבון GitHub, חשבון Railway (נרשמים דרך GitHub), ו-Node.js מותקן.
> אם משהו מזה חסר — Claude ינחה אותך להתקין/להירשם.

---

אני רוצה להקים לעצמי מערכת שליחת הודעות וואטסאפ מתוזמנות לקבוצות, מהתבנית המוכנה.
עשה בשבילי את כל העבודה הטכנית מקצה לקצה, ותעצור לשאול אותי רק בשלושת המקומות שחייבים אותי
(התחברות לחשבונות, הזנת תשלום, סריקת QR). אחרי כל שלב אמור לי בקצרה מה עשית. בוא נתחיל:

**פתיחה — הסבר לי ואסוף פרטים (לפני שמתחילים לעבוד)**
לפני כל דבר, אמור לי בבירור:
1. מה יידרש ממני עצמי במהלך התהליך: (א) להתחבר לחשבונות GitHub ו-Railway בדפדפן; (ב) להזין אמצעי תשלום ב-Railway (~5$ לחודש); (ג) לסרוק QR מהוואטסאפ בטלפון. כל השאר עליך.
2. שאל אותי ואסוף מראש את הפרטים הבאים להתאמה אישית, ורשום אותם לשימוש בהמשך:
   - מספר הוואטסאפ שממנו אחבר את הבוט (ל-NOTIFY_JID, לקבלת התראת התנתקות).
   - האם ארצה גם התראות במייל? (אם כן — כתובת מייל ו-Gmail App Password; אם לא — נדלג).
   - איך לקרוא לאפליקציה / שם ה-repo (ברירת מחדל: whatsapp-scheduler).
   - האם ארצה לקטלג את הקבוצות בקטגוריות בטופס? אם כן — אילו קטגוריות ולפי אילו מילות-מפתח בשמות הקבוצות (זה נמצא בפונקציה categoryOf ב-index.js, וניתן לערוך בקלות).
המתן שאענה על אלה לפני שתמשיך.

**שלב 0 — בדיקות מקדימות**
- ודא ש-git ו-Node.js מותקנים (`node -v`, `git --version`). אם לא — הנחה אותי להתקין.
- ודא שאני מחוברת ל-GitHub דרך `gh auth status`. אם לא — הרץ `gh auth login` והדרך אותי צעד-צעד בדפדפן.

**שלב 1 — הבאת הקוד**
- שכפל: `git clone https://github.com/meirdarkecha-dot/whatsapp-scheduler-template.git whatsapp-scheduler`
- היכנס לתיקייה והרץ `npm install`.
- צור לי repo פרטי חדש ב-GitHub בשם whatsapp-scheduler ודחוף אליו את הקוד (`gh repo create ... --private --source=. --push`).

**שלב 2 — פריסה ל-Railway (כאן נכנס ה-5$)**
- התקן Railway CLI: `npm i -g @railway/cli`.
- הרץ `railway login` ותן לי להתחבר בדפדפן.
- הסבר לי בקצרה: Railway גובה בערך 5$ לחודש על הרצה רציפה. הנחה אותי איך להוסיף אמצעי תשלום ולבחור בתוכנית (Account > Billing). חכה שאאשר שסיימתי.
- צור פרויקט: `railway init`, וחבר אותו ל-repo שלי.
- **הוסף שירות PostgreSQL לפרויקט** (זה קריטי — מזריק אוטומטית את DATABASE_URL ששומר את ה-session וההודעות כך שלא צריך לסרוק QR מחדש בכל דפלוי).
- הפעל דומיין ציבורי (Settings > Networking > Generate Domain) וקבל את הכתובת.

**שלב 3 — הגדרות**
- הגדר ב-Railway את משתני הסביבה:
  - `APP_URL` = כתובת הדומיין.
  - `NOTIFY_JID` = מספר הוואטסאפ שלי בפורמט `972XXXXXXXXX@s.whatsapp.net` (שאל אותי מה המספר). כך אקבל התראה **רק** אם הבוט מתנתק.
  - אם ביקשתי התראות מייל — הגדר גם `ALERT_EMAIL`, `GMAIL_USER`, `GMAIL_PASS`.
  - שאר המשתנים ב-.env.example — השאר ריקים (אופציונליים).
- אם ביקשתי קטגוריות מותאמות לקבוצות — ערוך את הפונקציה categoryOf ב-index.js לפי מילות-המפתח שנתתי, ודחוף מחדש.
- ודא שהדפלוי עלה: פתח `<APP_URL>/state` וודא שמחזיר סטטוס (יהיה qr בהתחלה — זה תקין).

**שלב 4 — חיבור הוואטסאפ (סריקת QR)**
- פתח לי בדפדפן `<APP_URL>/qr`.
- הדרך אותי: בטלפון → וואטסאפ → הגדרות → מכשירים מקושרים → קישור מכשיר → לסרוק את הקוד במסך.
- ודא שאחרי הסריקה `<APP_URL>/state` מחזיר `connected`.

**שלב 5 — הדרכה קצרה לשימוש**
- הסבר לי איך משתמשים:
  - **עמוד ראשי** = יצירת הודעה מתוזמנת (טקסט/תמונה, בחירת קבוצות, תאריך ושעה בשעון ישראל).
  - **/groups** = הוספת הקבוצות שאני חברה בהן לרשימת היעדים (הבוט חייב להיות חבר בקבוצה).
  - ההודעות וה-session נשמרים ב-PostgreSQL, אז redeploy או restart לא מוחקים כלום.

חשוב לאורך כל הדרך: אל תמחק או תדרוס הודעות מתוזמנות קיימות. אל תבצע פעולות שדורשות אמצעי תשלום או סיסמה בלעדיי — תמיד עצור ותן לי לעשות את החלק הזה. בסיום, ודא ש-`/state` מחזיר connected ותן לי סיכום קצר של מה שהוקם.
