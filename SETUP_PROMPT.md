# 🚀 נוסח להדבקה ב-Claude Code

מעתיקים את כל הבלוק שבין הקווים ומדביקים ב-Claude Code (בתיקייה ריקה חדשה).
Claude יבנה את הכל לבד — יישאר רק לסרוק QR מהטלפון ולהוסיף קבוצות.

> לפני שמתחילים צריך: חשבון GitHub (חינם), חשבון Railway (חינם, נרשמים דרך GitHub), ו-Node.js מותקן.

---

אני רוצה להקים לעצמי מערכת שליחת הודעות וואטסאפ מתוזמנות לקבוצות — אותה מערכת מהתבנית המוכנה.
בצע בשבילי את כל השלבים, ותנחה אותי רק במקומות שחייבים אותי (התחברות לחשבונות, סריקת QR):

1. שכפל את התבנית מ-GitHub:
   `git clone https://github.com/meirdarkecha-dot/whatsapp-scheduler-template.git whatsapp-scheduler`
   היכנס לתיקייה והרץ `npm install`.

2. צור לי repo חדש ופרטי ב-GitHub בשם whatsapp-scheduler ודחוף אליו את הקוד (השתמש ב-gh CLI; אם אני לא מחובר, הנחה אותי להתחבר).

3. פרוס ל-Railway:
   - התקן את Railway CLI (`npm i -g @railway/cli`) והנחה אותי ל-`railway login`.
   - צור פרויקט חדש (`railway init`), חבר אותו ל-repo, והוסף שירות **PostgreSQL** לפרויקט (זה מזריק אוטומטית את משתנה DATABASE_URL — קריטי לשמירת ה-session וההודעות).
   - הפעל דומיין ציבורי (Settings > Networking > Generate Domain) וקבל את הכתובת.

4. הגדר משתני סביבה ב-Railway:
   - `APP_URL` = כתובת הדומיין שקיבלת.
   - `NOTIFY_JID` = מספר הוואטסאפ שלי בפורמט `972XXXXXXXXX@s.whatsapp.net` (שאל אותי מה המספר) — לקבלת התראה רק כשהבוט מתנתק.
   - (השאר את שאר המשתנים ב-.env.example ריקים — הם אופציונליים.)

5. אחרי שהדפלוי עלה, פתח לי את הכתובת `<APP_URL>/qr`, ואני אסרוק את הקוד מהוואטסאפ בטלפון (הגדרות > מכשירים מקושרים > קישור מכשיר).

6. הסבר לי איך לנהל את המערכת:
   - כניסה לכתובת הראשית = יצירת הודעה מתוזמנת (טקסט/תמונה, בחירת קבוצות, תאריך ושעה).
   - `/groups` = הוספת הקבוצות שאני חברה בהן לרשימת היעדים.
   - ההודעות וה-session נשמרים ב-Postgres, אז redeploy לא מוחק כלום.

חשוב: אל תמחק או תדרוס הודעות מתוזמנות קיימות בשום שלב. ודא בסוף שהאפליקציה מגיבה (`/state` מחזיר connected) לפני שאתה מסיים.
