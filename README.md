# WhatsApp Scheduler — תבנית מוכנה

מערכת לשליחת הודעות וואטסאפ **מתוזמנות** לקבוצות (טקסט או תמונה), עם ממשק ניהול בעברית.
נבנתה עם Baileys + PostgreSQL, רצה על Railway.

## מה זה נותן
- 📅 תזמון הודעות לקבוצות לפי תאריך ושעה (שעון ישראל)
- 🖼️ טקסט או תמונה עם כיתוב
- 📁 ארגון קבוצות בקטגוריות בטופס הבחירה
- 💾 שמירה ב-PostgreSQL — שורד restarts ו-redeploys בלי לסרוק QR מחדש
- 🔔 התראה בוואטסאפ/מייל **רק** כשהבוט מתנתק
- 🛡️ הגנה מפני שליחה כפולה

## התקנה מהירה
פתחו את [SETUP_PROMPT.md](SETUP_PROMPT.md), העתיקו את הנוסח והדביקו ב-Claude Code — הוא בונה הכל.
או ידנית: ראו את השלבים ב-SETUP_PROMPT.md.

## משתני סביבה
ראו [.env.example](.env.example). היחיד שחובה הוא `DATABASE_URL` (מוזרק אוטומטית ע"י Postgres ב-Railway).

## שימוש יומיומי
- **עמוד ראשי** — יצירת הודעה מתוזמנת + צפייה בממתינות/שנשלחו
- **/groups** — הוספת קבוצות שהבוט חבר בהן לרשימת היעדים
- **/qr** — סריקת QR לחיבור/חיבור מחדש
- **/state** — בדיקת סטטוס (connected / qr / disconnected)

## הערה
Baileys הוא ממשק לא-רשמי לוואטסאפ. שליחה מרובה לקבוצות עלולה לעיתים לגרום לוואטסאפ לנתק את המכשיר. השתמשו באחריות ובקצב סביר.
